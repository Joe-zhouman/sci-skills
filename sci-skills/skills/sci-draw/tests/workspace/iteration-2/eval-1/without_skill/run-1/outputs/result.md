# Figure 1 Layout Plan for Nature Materials

## Overview

This figure presents the nanoparticle drug delivery system, covering formulation characterization, drug release kinetics, and biocompatibility. The layout follows a standard 2x2 grid for Nature Materials (full-page width ~183 mm, single-column ~89 mm).

---

## Panel Layout (2 x 2 Grid)

### Panel a -- Schematic Illustration (upper-left)
- **Content:** A cartoon/schematic of the nanoparticle drug delivery system, showing the core-shell structure, drug loading, and surface functionalization (if applicable).
- **Purpose:** Introduces the system visually and sets context for the reader. This is standard for Figure 1 in Nature Materials.
- **Size:** ~85 mm x 85 mm (square or slightly rectangular).
- **Style:** Clean vector illustration, muted color palette, no excessive detail.

### Panel b -- Nanoparticle Characterization (upper-right)
- **Content:** Two side-by-side bar charts or a grouped bar chart:
  - Left: Hydrodynamic size (nm) for 4 formulations (measured by DLS).
  - Right: Zeta potential (mV) for 4 formulations.
- **Purpose:** Demonstrates that the 4 formulations have distinct and well-characterized physicochemical properties.
- **Size:** ~85 mm x 60 mm.
- **Details:**
  - X-axis: Formulation labels (e.g., F1, F2, F3, F4).
  - Y-axis (left chart): Size (nm); Y-axis (right chart): Zeta potential (mV).
  - Error bars: Mean +/- SD (n >= 3).
  - Statistical annotations: asterisks or letters for significant differences (p < 0.05).
  - Consider a small table inset if the data is simple enough.

### Panel c -- In Vitro Drug Release (lower-left)
- **Content:** Line plot showing cumulative drug release (%) over 72 hours for all 4 formulations.
- **Purpose:** Shows sustained/controlled release behavior, which is the core functional claim of the delivery system.
- **Size:** ~85 mm x 60 mm.
- **Details:**
  - X-axis: Time (hours), 0-72 h.
  - Y-axis: Cumulative release (%), 0-100%.
  - One line per formulation (4 lines), with distinct colors or line styles.
  - Error bars at each time point (mean +/- SD).
  - Inset: zoomed view of early release (0-6 h) if needed to highlight burst release differences.

### Panel d -- Cell Viability (lower-right)
- **Content:** Grouped bar chart showing cell viability (%) at 3 concentrations for each formulation (or a single formulation vs. control).
- **Purpose:** Demonstrates biocompatibility / low cytotoxicity, a critical safety claim.
- **Size:** ~85 mm x 60 mm.
- **Details:**
  - X-axis: Concentration (e.g., low, medium, high; or specific ug/mL values).
  - Y-axis: Cell viability (%), 0-120% (with 100% dashed line for control).
  - Grouped by formulation (4 bars per concentration group) or by concentration (3 bars per formulation group).
  - Dashed horizontal line at 100% (untreated control).
  - Statistical annotations for significance vs. control.
  - If only one formulation was tested for viability, a simpler single-series bar chart suffices.

## Design Guidelines (Nature Materials)

| Aspect | Recommendation |
|---|---|
| **Figure width** | 183 mm (full page) or 89 mm (single column) |
| **Font** | Arial or Helvetica, 6-8 pt in final figure |
| **Color palette** | Colorblind-friendly (e.g., Okabe-Ito, viridis); avoid pure red/green |
| **Error bars** | Always show; specify n and error type in caption |
| **Panel labels** | Bold lowercase letters: **a**, **b**, **c**, **d** (top-left of each panel) |
| **Resolution** | 300 DPI minimum for raster; prefer vector (EPS/PDF) |
| **File format** | EPS or PDF for submission; TIFF acceptable |
| **Line weight** | 0.5-1 pt for axes, 1-2 pt for data lines |

---

## Suggested Caption

> **Figure 1 | Characterization and performance of nanoparticle drug delivery formulations.** **a**, Schematic illustration of the nanoparticle structure. **b**, Hydrodynamic size (left) and zeta potential (right) of four formulations (F1-F4) measured by dynamic light scattering. Data are mean +/- SD (n = 3). **c**, In vitro cumulative drug release profiles over 72 hours in phosphate-buffered saline (pH 7.4, 37 degrees C). **d**, Cell viability (MTT assay) after 24 h exposure at three concentrations. Dashed line indicates untreated control (100%). Data are mean +/- SD (n = 3); *p < 0.05, **p < 0.01 vs. control (one-way ANOVA with Tukey post-hoc test).

## Notes

- If the paper has a graphical abstract, panel (a) can be reused or simplified for it.
- Consider adding a small table in the characterization panel if the exact numeric values are important for the reader.
- The release curve panel is the strongest result -- place it prominently (upper-right would be ideal, but the schematic is conventional in that position for Figure 1).
- All figures should be prepared at final size; do not resize after creation.
